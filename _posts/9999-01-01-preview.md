## About functional js

### 비동기성

비동기 상황을 값으로 다룰 수 있다. Promise

### promise 일급 활용

```js
const delay100 = (a) =>
  new Promise((resolve) => setTimeout(() => resolve(a), 100));
// 동기적으로 실행이 되어야 한다 ??
const go1 = (a, f) => (a instanceof Promise ? a.then(f) : f(a));
const add5 = (a) => a + 5;

const n1 = 10;
const n2 = delay100(10);
go1(go1(n1, add5), console.log);
go1(go1(n2, add5), console.log);
```
